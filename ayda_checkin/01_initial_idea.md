# Ayda Check-in: Начальная идея

**Дата:** 2025-01-17

## Контекст

### Ситуация в Алматы
1. Туристы (особенно молодые) ходят в горы неподготовленными → несчастные случаи, вызов спасателей
2. Активный проект развития пешего туризма:
   - Строятся 3 новых "визит-центра" (сервисные центры для туристов)
   - План на год: обиходить 300 км троп (лавочки, указатели и т.д.)

### Возможность через Нияза Янзакова
- Один из основателей и тренер бегового клуба SRG
- Имеет связи с людьми, занимающимися визит-центрами и проектом 300 км троп
- Визит-центры предоставляют пространство для SRG и других участников/туристов
- Изначальный запрос: QR-коды для рекламы и заявок в клуб SRG

## Идея: QR Check-in система

### Концепция
QR-коды для чекина на маршрутах. Турист:
1. Сканирует QR-код на старте маршрута → чекинится что стартовал
2. На развилках, указателях, лавочках — дополнительные QR для чекина
3. Система замеряет скорость и положение туриста

### Ключевые преимущества
- **Работает при слабой связи** (офлайн с отложенной синхронизацией)
- **Безопасность**: известна последняя точка человека
- **Простота**: QR можно наклеить на любую поверхность
- **Масштабируемость**: универсальные или уникальные QR для каждой точки

## Решения по ключевым вопросам

### 1. Куда встраивать?
**Требует дополнительного анализа.** Варианты:
- ayda_run (существующая база, беговые клубы)
- gpx-predictor (синергия с прогнозированием)
- Отдельный продукт (лёгкий вход, своя ценность)

### 2. Авторизация
- НЕ анонимно (нужна идентификация для безопасности)
- Telegram (быстрый вход)
- Полная регистрация (опционально)

### 3. Бизнес-модель
- Бесплатно для пользователей (базовый функционал)
- Монетизация через визит-центры/гранты
- Возможность докупки (премиум функции?)

### 4. Масштаб
- Старт: Алматы
- Потенциал: расширение на другие регионы

## Синергия с существующими проектами

### GPX Predictor
- Реальные данные о прохождении маршрутов
- Калибровка модели на казахстанских тропах
- Функция "сколько осталось идти"

### Ayda Run
- Воронка пользователей через чекины
- Трейлы с авто-чекинами для беговых клубов
- Интеграция со Strava
